# LifeShieldV2

LifeShieldV2 is a mobile-first (Vite + TypeScript) single-page app with modular “islands” for different analytical approaches.

## Local development

```bash
npm install
npm run dev
```

## Build & preview

```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages

```bash
npm run build
npm run preview
```

Then publish the `dist/` folder with GitHub Pages (Settings → Pages → Build from a branch, `/dist`).
