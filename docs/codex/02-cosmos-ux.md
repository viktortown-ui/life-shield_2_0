# Cosmos UX — Prompt 2

## Что сделано

- Добавлен SVG-контейнер `g.cosmos-viewport`, в котором применяется общий `transform` (`translate + scale`) для карты.
- Реализована навигация через Pointer Events:
  - drag (мышь/палец) — pan;
  - wheel — zoom в точку под курсором;
  - pinch (2 касания) — zoom в центр жеста;
  - double-click / double-tap — reset view.
- Добавлены ограничения:
  - `minZoom = 1`, `maxZoom = 3.2`;
  - clamp для pan, чтобы нельзя было «улететь» слишком далеко.
- Добавлена кнопка `Сброс вида` поверх карты.
- Улучшена интерактивность планет:
  - невидимый `hit`-круг (`r=22`, минимум ~44px hit area);
  - выделение выбранной планеты через halo.
- Улучшены подписи:
  - по умолчанию видимы для 7 «важных» планет;
  - для остальных подписи показываются на hover (desktop) и при выборе (tap/click);
  - подписи ограничены до 2 строк;
  - подписи сдвигаются внутрь карты, если попадают к краю.
- Улучшено позиционирование контекстного меню:
  - меню открывается рядом с выбранной планетой;
  - меню клампится внутри контейнера;
  - при пересечении с подписью меню сдвигается.
- Подписи орбит приведены к русскому языку (`Финансы`, `Обязательства`, `Доход`, `Энергия`, `Поддержка`, `Гибкость`).

## Управление жестами

### Desktop
- Потянуть карту мышью — pan.
- Крутить колесо мыши над картой — zoom в точку курсора.
- Двойной клик по карте — сброс вида.
- Клик по планете — выбор + контекстное меню.

### Mobile / touch
- Потянуть одним пальцем — pan.
- Двумя пальцами (pinch) — zoom.
- Двойной тап — сброс вида.
- Тап по планете — выбор + контекстное меню.

## Ограничения MVP

- Реализован простой подход к «важным» планетам (первые 7 в текущей конфигурации).
- Разрешение пересечений подписей между собой не включает сложную физику/алгоритм раскладки.
- При сильном zoom-in панорамирование ограничено clamp-границами для сохранения контента в зоне видимости.
