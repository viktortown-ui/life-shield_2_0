# Cosmos MVP (Prompt 1)

## Что сделано
- Добавлен новый экран Home `Cosmos` с маршрутом `#/cosmos`.
- Добавлена минимальная настройка `homeScreen: 'shield' | 'cosmos'` (по умолчанию `shield`) в существующее состояние приложения.
- На экране настроек добавлен переключатель «Домашний экран: Shield | Cosmos».
- Маршрут `#/` теперь открывает выбранный Home (Shield или Cosmos).
- Старый Shield сохранён и доступен напрямую по `#/shield`.
- Для Cosmos реализована SVG-карта с 6 орбитами доменов и планетами существующих островов.
- На планетах показывается статус-бейдж (ok / risk / нет данных).
- По клику на планету открывается контекст-меню: «Открыть», «Данные» (только для доступных), «Закрыть».

## Как включить Cosmos как Home
1. Открыть `#/settings`.
2. В блоке «Интерфейс» выбрать `Cosmos`.
3. Перейти на `#/` — будет открываться Cosmos.

## Известные ограничения MVP
- Позиции планет статичны (без физической симуляции).
- Контекст-меню сделано простым popover, не marking-menu.
- Кнопка «Данные» сейчас доступна только для островов, использующих общий экран `#/finance`.
